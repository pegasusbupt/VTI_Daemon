1. Install PostgreSQL 9.1 + PostGIS 1.5
2. Install Git for Windows. (Go over the setup process http://help.github.com/win-set-up-git/)
3. Install Java and Eclipse J2EE(EGit plugin for eclipse as well).
4. Install Apache Tomcat (http://tomcat.apache.org/tomcat-7.0-doc/setup.html#Windows)
integrate with eclipse (install web tools from install software): http://www.mulesoft.com/tomcat-eclipse
5. Clone git repository (ssh version of url) and import the project in Eclipse.
